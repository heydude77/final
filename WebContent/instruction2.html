(서버측에서 클라이언트에게 실시간 정보를 보내주기 위해선, WebSocket을 사용할수 있어야 한다) WebSocket 기술은 스크립트 기술임 (Ajax 가 발전된 기술) ──────────────────────────────────────── final 에 사원 로그인 기능 만들려고 함. (로그인을 하지 않으면 이 서비스는 사용불가) ▶ /index.do 를 매핑잡고, 로그인 폼을 구현 (web.xml 에 welcome-file-list 에 index.do 추가) (타일즈 처리 할필요 없고, bootstrap 이용해서 꾸며둔 독립 jsp 로 / id, pass만 적게) action을 login.do 로 ▶ /login.do 매핑 잡고 전달받은 값을 토대로 데이터베이스 작업을 완료, 세션 처리할꺼 하고, /index.do 로 리다이렉트 (※ /index.do 가 세션인증 받았냐 안받았냐에 따라서 view 가 다르게 구현이 되어야 함) ▶ /index.do 에서 인증받은 사용자의 뷰는 tiles로 처리 템플릿은 알아서, bootstrap 에서 example을 참조 하거나, admin-template 만들어둔거로 참조 해도 됨. ──────────────────────────────────────── ▶1# 로그인한 사용자가 항상 볼수 밖에 없는 뷰영역에 websocket 연결을 걸어두고, onmessage 잡아서 alert 를 받을수 있게 구현 ▶2# 어떤 사원이 로그인을 성공하게 되면, 연결되어 있는 모든 소켓에게 사원정보를 mode값을 적절하게 설정해서 sendMessage ▶3# onmessage 걸어둔 부분에 , 2번에서 설정한 모드값이 날라오면, 거기서 정보 뽑아서 HTML 만들어서 innerHTML 혹은 / window.alert() 로 어떤 사원이 로그인했다는 알수 있게 처리.
